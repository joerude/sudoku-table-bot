# 🧩 Sudoku League Bot

Телеграм-бот, который ведёт учёт соревновательной игры в судоку
([usdoku.com](https://www.usdoku.com/)) между друзьями: очки, места, сезоны —
вместо ручного Google Sheets.

## Возможности

- **Регистрация игроков** — `/join`
- **Создание игры** — `/newgame medium hardcore` → ссылка на usdoku + ожидающая игра
- **Запись результата в пару тапов** — `/result`, выбираешь игроков в порядке финиша
- **Очки** — настраиваемая таблица на сезон (дефолт `3 / 1 / 0`)
- **Сезоны** — до порога (дефолт 100 очков), затем авто-архив и старт нового
- **Статистика** — `/status`, `/season`, `/me`, `/history`
- **Напоминания** — ловит незаписанные игры

## Быстрый старт (Docker)

1. Получи токен у [@BotFather](https://t.me/BotFather).
2. Скопируй конфиг и впиши токен:
   ```bash
   cp .env.example .env
   # отредактируй BOT_TOKEN в .env
   ```
3. Запусти:
   ```bash
   docker compose up -d --build
   ```
4. Добавь бота в групповой чат и напиши `/start`.

Данные лежат в `./data/sudoku.db` (примонтированный volume), переживают перезапуск.

## Локальный запуск (без Docker)

```bash
export BOT_TOKEN=...   # токен от BotFather
export DB_PATH=./data/sudoku.db
go run ./cmd/bot
```

## Команды

| Команда | Описание |
|---------|----------|
| `/join [имя]` | Зарегистрироваться |
| `/players` | Список игроков |
| `/newgame <сложность> [режим]` | Новая игра + ссылка на usdoku |
| `/result` | Записать результат (тап по порядку финиша) |
| `/status` | Таблица текущего сезона |
| `/season` | Инфо о сезоне |
| `/me` | Личная статистика |
| `/history` | Последние игры |
| `/settings` | Настройки (только админ) |
